## 📋 Sobre

Proffy é um sistema de ensino onde alunos se conectam com professores.

----

## 💻 Tecnologias utilizadas

- React
- React Native
- Typescript
- Express
- Axios

## ⚙ Como executar

```bash

# backend
$ npm i
$ npm run migrate:latest

# web
$ npm i

# mobile
$ npm i
$ npm start 

```
